Git Commands!

  • git init: Give access to git to the current directory
  • git add . : adds all files in the current directory
  • git commit -m "first commit": to commit the changes
  • git branch -M main: initializing main branch
  • git remote add origin https://github.com/Saswata-2023/week1.git(link): specify the folder where you want to push
  • git push -u origin main: to push your main branch
  • git status: to view the current status
  • git pull: to sync the changes
